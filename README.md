## Local development

In order to develop and build this project, you will need [Gatsby](https://www.gatsbyjs.com/) installed.

- **Develop**: `gatsby develop`
- **Build**: `gatsby build`

## Theme

[Gatsby-theme-portfolio](https://github.com/smakosh/gatsby-theme-portfolio)
